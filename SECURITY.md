# Security Policy

## Supported Versions

This is a portfolio demo project. Security fixes target the current `master` branch only.

| Version | Supported |
| ------- | --------- |
| master  | Yes       |
| others  | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in this project:

1. Do not open a public issue.
2. Email 2054469475@qq.com with details.
3. Allow up to 7 days for an initial response.

## Scope

- `.env` file exposure
- API key or token leakage in commits, logs, or documentation
- Hardcoded credentials in source code
- Insecure dependency versions

## Out of Scope

- Third-party service vulnerabilities
- Third-party package vulnerabilities
- Phishing or social engineering attacks
